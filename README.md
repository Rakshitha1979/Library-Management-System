# 📚 Library Management System
A desktop application built using **Java Swing**, **JDBC**, and **MySQL** to simplify basic library operations through an easy-to-use graphical interface.
## Features
- Add New Books
- Display Available Books
- Issue Books
- Return Books
- Clear Input Fields
- Exit Application
## Technology Stack
- Java
- Java Swing
- JDBC
- MySQL
- Visual Studio Code
## Database
**Database:** `librarydb`
**Table:** `books`
## Getting Started
1. Open the project in Visual Studio Code.
2. Start the MySQL Server.
3. Create the `librarydb` database and `books` table.
4. Update the database credentials in `LibraryDB.java`.
5. Add the MySQL Connector JAR to the project.
6. Run `LibraryManagement.java`.
## Project Overview
This project demonstrates the implementation of:
- Object-Oriented Programming (OOP)
- Java Swing GUI
- JDBC Database Connectivity
- CRUD Operations
- Input Validation
**Developed as a Java Desktop Application for Library Management.**
